### 5. Тестирование 

С помощью команды python.by и с включенным VPN запустила бота.
![Запуск бота](../images/python2.jpg)

Проверила работу бота - все отлично работало (местами долго грузило). Записала видео-демо: https://drive.google.com/file/d/1MolXKYOo9Ttn0XW3sWzBb_nBN015nt7r/view?usp=sharing

Бот интегрирован с API и может показывать данные о погоде в 5 городах по команде. Бот отправляет запрос к Open-Meteo API и получает актуальную погоду. Помимо этого с помощью интеграции с CSV файлом бот может показывать данные о сотрудниках, собирать статистику. Бот читает файл data.csv с сотрудниками, ищет и фильтрует информацию.

Из библиотек используется python-telegram-bot 20.7, pandas 2.0.3, requests 2.31.0.
