### 3. Составление промпта для LLM

Дополнила промпт из инструкции другими требованиями из инструкции. Вставила в Cursor следующий промпт:

```text
                                                  
Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot (версия 20.x).

Функционал бота:
- Бот собирает анонимные мнения команды по текущему вопросу
- Пользователи могут поставить оценку от 1 до 5 командой /rate 5
- Пользователи могут оставить текстовый отзыв командой /feedback Текст отзыва
- Руководитель (заданный в переменной ADMIN_ID) может:
  - Создать новый опрос командой /new_question
  - Получить сводный отчёт командой /report
  - Получить статистику по всем опросам командой /stats
- Бот хранит данные в SQLite
- Бот отправляет напоминание неответившим пользователям по команде /remind (только руководитель)

Команды бота:
/start - приветствие и регистрация
/rate [1-5] - поставить оценку текущему опросу
/feedback [текст] - оставить текстовое мнение
/report - получить отчёт (только руководитель)
/stats - общая статистика (только руководитель)
/new_question - создать новый опрос (только руководитель)
/remind - напомнить неответившим (только руководитель)

Как бот должен отвечать:
- На /start: "Привет! Я бот для сбора обратной связи..."
- На /rate: "Спасибо! Твой голос учтён. Текущая средняя оценка: X"
- На /report: выдать отчёт в формате:
  Опрос: "Как прошёл спринт?"
  Период: 01.04 - 07.04
  Ответов: 15/20 (75%)
  Средняя оценка: 4.2
  Топ проблем (из feedback):
    1. Много багов
    2. Не хватает времени

Какие данные хранить:
- Таблица users: user_id, username, name, role (user/admin)
- Таблица surveys: id, question_text, start_date, end_date, active
- Таблица responses: id, user_id, survey_id, rating, feedback_text, created_at

Требования:
- Код должен быть простым, понятным, с комментариями
- Использовать python-telegram-bot версии 20.x (async)
- Добавить обработку ошибок (try-except)
- Использовать .env файл для токена и ADMIN_ID
- Бот должен работать при локальном запуске

Создай:
1. Файл bot.py — полный код бота
2. Файл requirements.txt — зависимости
3. Файл .env.example — пример конфигурации
4. Файл README.md — инструкция по запуску

```
